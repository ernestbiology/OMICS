## Проект 1: Hi-C
 
Сравнение контактных карт Hi-C для образцов **MoPh7, MoPh11, MoPh14, MoPh15**.
 
**Вывод:**
> Полногеномные карты показывают ожидаемый диагональный паттерн контактов, явных крупных межхромосомных перестроек не выявлено.
 
---
 
## Проект 2: RNA-seq / CAGE
 
Построены треки покрытия RNA-seq (на основе **STAR**) и **CAGE**, проведено сравнение в IGV.
 
```
Локус: chr3:99.31–99.34 Mb
```
 
**Вывод:**
> RNA-seq и CAGE показывают сигнал, ассоциированный с транскрипцией, в выбранном локусе. STAR подходит для выравнивания RNA-seq лучше, чем BWA, так как учитывает сплайсинг.

## Проект 3: ChIP-seq
 
Обработаны данные ChIP-seq для меток:
 
- `H3K9me3`
- `H3K27Ac`
- `H2A119Ub`
- `CTCF`

 
**Вывод:**
> CTCF показывает выраженное обогащение относительно input; **MACS3** выявляет узкий пик с чёткой вершиной (summit).
